## 기술 스택 및 선택 이유

### FrontEnd

- `React`
  - React 기반 컴포넌트의 반응형 동작 방식을 학습하기 위해
  
- `Next.js`
  - 추후 SSR를 통해 포트폴리오 데이터를 받아오도록 확장을 고려하여 선택

### CSS

- `TailwindCSS`
  - className기반의 CSS 도구를 활용해 보기 위해

### Animation

- `GSAP`
  - 반응형 스크롤 애니메이션 구현을 위해
